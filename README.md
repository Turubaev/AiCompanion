В) Составление другим ИИ (DeepSeek) промпта для решения задачи
Ты виртуальный ассистент. Тебе нужно решить задачу, следуя инструкциям:

Контекст задачи:
- Инструменты: Два пустых ведра. Ёмкость первого (большого) — 5 литров. Ёмкость второго (малого) — 3 литра.
- Ресурс: Неограниченный источник воды.
- Ограничения: Для манипуляций можно использовать только эти два ведра. Нельзя использовать маркировку, глазомер или сторонние предметы.
- Целевое состояние: В одном из вёдер должно быть ровно 4 литра воды.

Твоя задача:
Предложи последовательность чётких шагов (переливаний, наполнений, опорожнений), которые приведут к целевому состоянию.
Формат вывода: Пронумерованный список действий. В конце укажи, в каком ведре оказалось 4 литра.
Пример формата шага: "1. Наполнить большое ведро (5 л) полностью."

Г) Создание внутри ИИ группы экспертов, для решения задачи
Я хочу решить логическую задачу, используя метод "совета экспертов". Создай виртуальную группу из 4 экспертов с разными подходами к решению задач.

Задача:
Есть 2 пустых ведра: 5-литровое и 3-литровое. Неограниченный запас воды. Можно только наливать, выливать и переливать. Цель: получить ровно 4 литра в одном из вёдер.

Инструкции для каждого эксперта:
1. Эксперт-алгоритмист: Думай как программист. Рассмотри задачу как поиск в пространстве состояний. Предложи наиболее эффективный алгоритмический путь.
2. Эксперт-математик: Используй модульную арифметику и комбинаторику. Объясни решение через уравнения и остатки от деления.
3. Эксперт-практик: Думай "руками". Представь, что ты реально переливаешь воду. Какие действия интуитивно приведут к цели?
4. Эксперт по креативным решениям: Рассмотри неочевидные подходы. Может быть, есть несколько разных путей к ответу?

Формат вывода:
Для каждого эксперта:
- Имя и специализация
- Краткая логика подхода
- Пошаговое решение
- Итоговое состояние вёдер

После всех решений напиши краткий итог: все ли эксперты пришли к одному ответу? Какой путь самый эффективный?
